Viikko 2: Hyviä ohjelmointikäytäntöjä – tyyppiturvallisuuden tavoittelua

Javaskriptin datatyypit ovat:

  • String
  • Numero
  • Boolean
  • Objekti
  • Funktio

Lisäksi on datatyypit, jotka eivät voi sisältää arvoja:

  • null
  • undefined

Objekteja on:

  • Objekti
  • Päivämäärä
  • Array

Onko objekti numero:

function isNumber(obj) { return !isNaN(parseFloat(obj)) }

Onko objekti array?

return myArray.constructor.toString().indexOf("Array") > -1;

}

Onko objekti päivämäärä?

function isDate(myDate) { return myDate.constructor.toString().indexOf("Date") > -1; }
